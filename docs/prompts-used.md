# Prompts utilizados con IA

## Estructura del proyecto

"Estoy creando una aplicación del clima usando HTML, CSS y JavaScript. ¿Puedes sugerir una estructura de carpetas clara y organizada?"

---

## Consumo de API

"Usando JavaScript y Fetch API, escribe una función que obtenga datos meteorológicos desde Open-Meteo usando geocodificación."

---

## Caché

"Escribe una función que almacene en caché los resultados del clima durante una hora usando localStorage."

---

## Diseño

"Sugiere mejoras en CSS y diseño para que mi app del clima sea moderna y responsive."
