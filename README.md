# Atmosfera

Atmosfera es una aplicación web desarrollada con HTML, CSS y JavaScript que permite consultar el clima actual de cualquier ciudad utilizando la API de Open-Meteo.

---

## Funcionalidades

- Buscar clima por ciudad
- Obtener temperatura actual
- Mostrar velocidad del viento
- Manejo de errores
- Caché con localStorage
- Diseño responsive
- Consumo de APIs usando Fetch

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Open-Meteo API
